Welcome to Astrix!
==================================

Astrix (AStrophysical fluid dynamics on TRIangular eXtreme grids) is a
CUDA/C++ implementation of a two-dimensional residual distribution scheme aimed at
tackling problems in astrophysical fluid dynamics.
